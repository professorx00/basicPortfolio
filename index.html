David Ward is a SQL Report and Python Developer who works with CTR Maine to create Desktop and Web Service interfaces for a hosted Time and Attendance Software using SOAP API and Direct File Manipulation.


David knows how to create python scripts to create interface structures to convert files into importable data formats including date/time stamp management, data processing, and graphing data.


David has developed crucial relationships with clients which has gained increase revenue for the CTR Maine. David even stepped into the secondary side of the business of security by taking incoming SQL Reports and creating a time sheet for employees based on their actual entrance and exit of the building.


David on his own time is taking ownership of his education. He is currently attending University of New Hampshire bootcamp for Full Stack Developer. David also producing 3D assets for Maya, Blender and 3D Max on a monthly basis.


On top of the education list above, David has an associate degree in Character Animation from Brooks College. David has also taken course from Animation Mentor where he was taught Python to enhance his ability with animation by using scripting.
